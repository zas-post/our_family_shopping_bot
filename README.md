# Family Shopping List Telegram Bot 🛒

Асинхронный Telegram-бот для ведения совместного семейного списка покупок в группах. Бот автоматически закрепляет актуальный список вверху чата и «тихо» обновляет его при добавлении или удалении товаров, не перегружая семейный чат лишними уведомлениями.

## ✨ Основные фичи

* **Интеллектуальный парсинг:** Автоматически распознает количество и единицы измерения товара (шт, кг, л, уп и т.д.) в конце строки с помощью регулярных выражений.
* **Автозакрепление и тихие обновления:** Бот держит список покупок в закрепе группы. Добавление новых позиций через префикс `п+` бесшумно обновляет инлайн-кнопки в закрепленном сообщении, а само сообщение пользователя удаляется для поддержания чистоты чата.
* **Интерактивый интерфейс:** Вычеркивание продуктов в один клик по инлайн-кнопке.
* **Оповещение «Я в магазине!»:** Специальная кнопка, которая отправляет громкий алерт всей семье о том, что список скоро «заморозится». Защищена встроенным анти-спам механизмом (cooldown 15 минут).
* **Коммерческая архитектура:** Проект полностью переведен на модульную структуру с использованием `Router`, изолированным слоем утилит и безопасным управлением конфигурацией через переменные окружения.

## 🛠 Стек технологий

* **Python 3.10+**
* **Aiogram 3.x** (Асинхронный фреймворк для Telegram Bot API)
* **SQLAlchemy 2.0** (ORM для работы с базой данных)
* **Alembic** (Инструмент для контроля и автоматической генерации миграций БД)
* **SQLite / Aiosqlite** (Легковесная асинхронная база данных)
* **Python-dotenv** (Управление секретами и конфигурацией)

## 🚀 Быстрый запуск

### 1. Клонирование и подготовка окружения
Создайте виртуальное окружение и активируйте его:

```Bash
python -m venv .venv
source .venv/bin/activate  # Для Linux/macOS
# или
.venv\Scripts\activate     # Для Windows
```

2. Установка зависимостей
Установите все необходимые пакеты одной командой:

```Bash
pip install -r requirements.txt
```

3. Настройка переменных окружения
Скопируйте шаблон конфигурации и создайте личный файл .env:

```Bash
cp .env.example .env
Откройте созданный файл .env и вставьте ваш токен бота, полученный от @BotFather:

BOT_TOKEN=ваша_строка_токена_сюда
```

4. Применение миграций базы данных
Создайте и обновите структуру таблиц в базе данных до актуального состояния с помощью Alembic:

```Bash
alembic upgrade head
````

5. Запуск бота

```Bash
python main.py
```

## ⚙️ Настройка прав бота в Telegram-группе

Для того чтобы фичи автозакрепа и очистки чата работали корректно, добавьте бота в вашу семейную группу и наделите его правами Администратора со следующими включенными разрешениями:

**Удаление сообщений (Delete messages)** — чтобы бот мог стирать строчки добавления продуктов (п+ ...).

**Закрепление сообщений (Pin messages)** — чтобы бот мог вешать и обновлять список покупок в самом верху чата.


## 📝 Примеры использования

Добавление одного товара: п+ Молоко 2л

Добавление нескольких товаров через запятую: п+ Хлеб, Сыр 300гр, Бананы 1кг

Вычеркнуть/вернуть товар: Нажмите на кнопку с названием товара в списке.

Полная очистка: Нажмите кнопку 🗑 Очистить купленное под списком.

Оповестить семью: Нажмите кнопку 🏃‍♂️ Я в магазине!.
